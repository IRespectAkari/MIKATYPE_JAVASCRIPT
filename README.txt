美佳のタイプトレーナ Ｖｅｒ２．０６．０３ JAVASCRIPT版ソースコード

　教育用のタイプ練習ソフト「美佳のタイプトレーナV2.06」のJAVASCRIPT版ソースコードです。
　ブラウザーで作動します。
　Chrome、Edge、Firefoxで動作確認済です。
　Chromebookでも動作します。
　Chromebookでは機種によっては極まれにフリーズする事がありますが、ChromeOSの最新バージョン120では不具合は解消されています。
　著作権を表示する初期画面は表示されません。
　成績はクッキーに保存されます。保存期間は最大で一年です(ブラウザーの使用状況によっては一年以内に成績が消去されることがあります)。
　成績はサーバーには保存されません。
　オフラインでも作動しますがオフラインで実行した場合は成績は保存されません。
　Windows版では練習時間記録ファイルmikatype.logと最高速度記録ファイルmikatype.spdが保存されますが、
　このJAVASCRIPT版ではmikatype.logとmikatype.spdは保存されません。
　Windows版では合計練習時間はプログラムを起動してから終了するまでの時間ですがJAVASCRIPT版では個別の累積練習時間の合計です。
　Windows版では実行時にウィンドーのサイズを変更できますが、このJAVASCRIPT版では実行時にサイズの変更はできません。
　ウィンドーのサイズを変更したい場合はindex.htmlのキャンバスサイズを変更して下さい。
　メニューの項目に「終了」はありません。終了したいときはウィンドーをクローズして下さい。

　このソースコードの著作権は今村二朗が所有します。
　改変、転載、他ソフトの使用など自由にお使いください。
